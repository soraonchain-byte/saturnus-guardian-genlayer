# 🪐 Saturnus Guardian

> **Architecting Cognitive Guardrails for the Agentic Economy.**

Saturnus Guardian is an **Intelligent Contract (IC)** built on **GenLayer Testnet Bradbury**. It addresses the "Clarity Gap" in Web3 wallet security by using Subjective Consensus to validate transaction intent before execution.

---

## 🏗️ Architecture
Saturnus Guardian functions as a middle-layer coordination infrastructure. It doesn't just verify signatures; it interprets the **narrative intent** of a transaction.

### Key Components:
- **Intent Analyzer**: Parses transaction metadata using GenLayer's LLM-powered logic.
- **Subjective Validator**: Triggers a Trustless LLM Jury to evaluate risk based on user-defined safety narratives.
- **Gatekeeper Module**: Blocks "drainer" or "sweeper" patterns that traditional deterministic contracts might miss.

---

## 🚀 Deployment (Bradbury Testnet)
The contract is written in Python using GenLayer's IC standards.

- [x] Basic security level configuration.
